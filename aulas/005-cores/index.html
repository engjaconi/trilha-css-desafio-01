Cores

Pré Definidas

Cores em inglês, não recomendado, pois pode variar de navegador para navegador.

color: red

RGB

Valores de 0-255, primeiro valor vermelho, segundo azul e terceiro verde.

color: rgba(0, 0, 0)

RGBA

Valores de 0-255, primeiro valor vermelho, segundo azul e terceiro verde. Quarto valor é o canal alpha, transparência de 0-1.

color: rgba(0, 0, 0)

HEXADECIMAL

Valores de 0-F. Adicionando mais dois valores ao final será o canal alpha.

color: #FFFF0A

HSL

Primeiro valor de 0-360. Verde de 0-120, Azul de 121-240 e Vermelho 241-360. Segundo é a saturação de 0-100%. Terceiro valor é a luminosidade.

HSLA, seria para passar o canal alpha.

color: hsl(120, 100%, 50%)
