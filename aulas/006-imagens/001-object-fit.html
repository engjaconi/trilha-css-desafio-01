Object Fit

Determina como será ajustado a imagem ou video, preservando a proporção ou distorcer para ocupar o espaço.

Propriedade object-fit: fill (padrão)

A imagem será distorcida.

Propriedade object-fit: contain

Imagem mantém a proporção, sendo diminuida para caber no espaço. Se for ampliada pode perder a qualidade.

Propriedade object-fit: cover

Imagem será cortada para preencher o espaço.

Propriedade object-fit: none

Imagem mantem o tamanho original e só exibe a área do container.

Propriedade object-fit: scale-down

Aplica o valor none ou contain e usa a que deixa a imagem menor.
